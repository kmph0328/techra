# TECHRA — 産業技術リテラシーの地図

社会人が技術ニュース・業界レポート・政策資料を読むための「産業技術リテラシー」を、
**用語の点ではなく、産業・工程・規制・市場の関係性の面**で学べる自律成長型学習サイトです。

## 起動方法

ビルド・サーバー不要です。`index.html` をブラウザで開くだけで動作します(オフラインでも動作)。

```
start index.html        # Windows
```

## Web公開と最新情報の自動反映

GitHub Pagesへの公開手順と、毎日ニュースを自動収集してサイトを成長させる
GitHub Actionsパイプライン(収集のみ/AI起草の2段階)は **[DEPLOY.md](DEPLOY.md)** を参照してください。

- ホスティング時は訪問のたびに `content/manifest.json` をチェックし、新版があれば
  `content/update.json` を取り込んでその場で再描画(本体の再配布なしに全訪問者へ反映)
- 取り込んだ差分はキャッシュされ、オフライン時も最後に同期した内容で表示
- パーソナライズは訪問者ごとにブラウザ内で独立動作(サーバー側のユーザー管理は不要)

## 主な機能

| 機能 | 説明 |
| --- | --- |
| 行動推定エンジン | 閲覧・滞在時間・節の展開・クイズ結果から「学習レンズ」(はじめて/技術/事業/規制)と関心領域を推定し、説明の順序・深さ・導線が自動で変化。手動設定は不要(ヘッダーのレンズチップで上書きも可能) |
| 多面的用語解説 | 28テーマを「一言定義・なぜ重要か・工程設備・規制・市場・リスク・誤解・指標・出典」で解説。本文中の用語は相互リンク+ホバー定義 |
| ニュース×学習 | 各ニュースに「前提知識チェックリスト」「観点別解説(初学者/技術/ビジネス/規制/リスク)」「事実と推測の区別」「今後の注目点」を付与。学習ページ⇄ニュースを双方向接続 |
| 知識マップ | 既読テーマが灯り、関連線がつながるSVG可視化。各用語ページにもミニマップ |
| 学習ルート | 目的別の体系学習順路(進捗追従・次の一歩の提示) |
| 復習エンジン | 既読・経過日数・誤答から復習対象を自動選定。クイズセッション・誤解チェック |
| 音声解説 | Web Speech APIで各テーマの「3分解説」(全体像→要点→誤解→復習)を自動生成・読み上げ |
| 自律成長の可視化 | 知識体系の更新履歴(成長ログ)、ニュース頻度から検知した新規テーマ候補、情報鮮度ダッシュボード |

## プライバシー

学習データ(閲覧履歴・推定された関心・クイズ結果)はブラウザの localStorage にのみ保存され、
外部送信は一切ありません。「成長ログ」ページからいつでもリセットできます。

## 構成

```
index.html          … SPAシェル
css/style.css       … デザインシステム
js/data/            … 知識データ(領域・用語・比較・ニュース・成長ログ)
js/sync.js          … オンライン更新(content/update.jsonの取込・キャッシュ)
js/usermodel.js     … 行動推定エンジン(localStorage、エクスポート/インポート対応)
js/audio.js         … 音声解説(speechSynthesis)
js/graph.js         … 知識マップ描画(SVG)
js/app.js           … ルーター・適応表示・各ページ描画
content/            … 配信コンテンツ(manifest.json + update.json + inbox.json)
tools/              … 自動更新スクリプト(RSS収集 + Claudeによる解説起草)
.github/workflows/  … Pagesデプロイ + 日次コンテンツ更新
```

## コンテンツの追加方法

`js/data/terms-*.js` に同じスキーマでオブジェクトを `TERMS.push({...})` するだけで、
ライブラリ・知識マップ・検索・復習・音声解説のすべてに自動反映されます。
ニュースは `js/data/news.js`、比較は `js/data/comparisons.js`、
成長ログ・新テーマ候補は `js/data/growth.js` を編集してください。
